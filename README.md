# CI/CD Pipeline with GitHub Actions + AWS EC2

## What this project does
Automatically deploys a Node.js app to AWS EC2 whenever 
code is pushed to the main branch.

## Architecture
GitHub → GitHub Actions → SSH → EC2 → PM2

## Tools Used
- Node.js + Express
- GitHub Actions
- AWS EC2 (Ubuntu)
- PM2 process manager
